## ✨ MPV 核心技术特性

| 分类 | 特性 |
| :--- | :--- |
| **渲染器** | `gpu-next`，支持 Vulkan / OpenGL / D3D11 后端 |
| **色彩管理** | ICC 配置 / 3D LUT 校准 / BT.709 / BT.2020 精确映射 |
| **杜比视界** | 杜比视界：Profile 5 / 8.1 / 7[FEL](https://github.com/mpv-player/mpv/pull/17932)）完整支持，合成BL+EL+RPU实现12bit色深处理链 |
| **HDR** | 峰值亮度自适应，支持 HDR10+ 直通 |
| **算法** | EWA Lanczos / FSRCNNX / Anime4K / RAVU |
| **硬件解码** | DXVA2 / D3D11VA / Vulkan / VAAPI / VDPAU 全后端 |
| **扩展性** | Lua 脚本 API，定制播放逻辑、界面、补帧等 |
| **AI 字幕** | 提取音频 → Whisper 本地识别 → SRT 源字幕 → 智谱 AI 翻译 → 双语字幕 → 自动加载 |

## 📌 关于建立本群的初衷想法

本群是为国内 MPV 爱好者建立的免费交流群，初衷很简单：

- 国内关于 MPV 的论坛和讨论区太少，大家遇到问题交流不便
- 很多新手不熟悉 GitHub 或英文，上手门槛太高
- 希望为中文用户提供一个互助、分享、共同进步的空间

群内所有整理和分享的内容，均会注明原作者并附上对应的 GitHub 仓库链接，尊重每位开发者的劳动成果。
